# Swarm

Survivor-like roguelite game developed in Unity (URP 2D).

## 🎮 About

This project aims to recreate and expand on the "Survivor-like" genre (e.g., Vampire Survivors), focusing on:

* Scalable enemy spawning system
* Upgrade-based progression
* Auto-attack mechanics
* Performance with large enemy counts

## 🧱 Tech Stack

* Unity 6 (URP 2D)
* C#
* GitHub for version control

## 📂 Project Structure

Assets/
Scripts/
Prefabs/
Scenes/
Art/

## ⚙️ Setup

1. Clone repository:

```bash
git clone https://github.com/uiuicorp/swarm.git
```

2. Open with Unity Hub (version 6000.x)

3. Open `SampleScene`

## 🚨 Rules for Collaboration

* Do NOT commit:

  * Library/
  * Temp/
  * Logs/

* Always:

  * Pull before starting work
  * Use branches
  * Commit small changes

## 🌱 Branch Strategy

* main → stable
* dev → integration
* feature/* → new features

## 💬 Commit Pattern

* feat: new feature
* fix: bug fix
* refactor: code improvement

Example:

```bash
git commit -m "feat: player movement system"
```
